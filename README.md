# Cronômetro digital 

- Cronômetro funcional, com opção de parar o contador, mantendo o tempo registrado e também zerar o contador, caso o mesmo esteja com algum valor não sendo o inicial.

## Tecnologias:
- HTML / CSS
- ReactJS
- Vite
